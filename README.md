# SS
SS는 System Scan의 약자로, 접속자의 네트워크 신호를 정밀하게 스캔하여 현재 위치와 통신 환경을 분석하는 서비스입니다.
## 🔎 Project: SC (System Scan)
사용자의 네트워크 신호를 분석하여 현재 위치와 통신 환경을 AI가 리포트 형식으로 들려주는 웹 애플리케이션입니다.

### ✨ Key Features
* **Real-time Scan**: 접속 즉시 IP, 도시, 통신사(ISP), 우편번호 등을 정밀 분석합니다.
* **AI Briefing**: 단순한 표 데이터가 아닌, 사람처럼 친절한 문장으로 정보를 요약하여 보고합니다.
* **Step-by-Step UI**: 정보 브리핑을 먼저 확인한 뒤, '다음' 버튼을 통해 실제 지도를 로드하는 단계별 인터페이스를 제공합니다.
* **Interactive Map**: 분석된 위도와 경도를 바탕으로 실시간 구글 지도를 시각화합니다. (무료 임베드 방식)

### 🛠 Tech Stack
* HTML5 / CSS3 / JavaScript (Vanilla JS)
* ip-api (Location Data)
* Google Maps Embed API (Visual Map)
